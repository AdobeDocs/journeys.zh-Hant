# 歷程欄位 {#sharing-journey-fields}

此Mixin用於歷程結構描述（與journeyStepEvent相關）。 它包含下列欄位：

## journeyID

主要歷程的ID。

類型: 字串

## journeyVersionID

歷程版本的ID。 此ID代表歷程的身分。

類型: 字串

## 名稱

歷程的名稱。

類型: 字串

## 說明

歷程的說明。

類型: 字串

## version

版本，表示為 `major`.`minor`

類型: 字串
