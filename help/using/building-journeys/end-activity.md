# 結束活動{#section_vqp_4ft_dgb}

此活 **[!UICONTROL End]** 動可讓您標籤每個歷程路徑的結尾。 它不是強制性的，但建議使用，以清楚顯示。 事實上，如果旅程有數個結束活動，我們建議您在每個結尾新增標籤，讓報表更容易閱讀。 請參閱[本頁](../reporting/about-journey-reports.md)。
