# journeyStep 事件識別欄位 {#sharing-identity-fields}

此Mixin專用於journeyStepEvent：此事件與歷程有關，沒有identityMap，無法描述設定檔身分（若有）。

若為journeyStepEvent，我們還需要新增與身分相關的欄位：

## profileID

設定檔識別碼

類型: 字串

## profileNamespace

設定檔識別碼名稱空間

類型: 字串
