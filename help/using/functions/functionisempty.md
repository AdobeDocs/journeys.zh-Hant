# IsEmpty {#isEmpty}

如果引數中的字串沒有字元，則傳回true。

## 類別

字串

## 函式語法

`isEmpty(<parameters>)`

## 參數

* 字串

## 簽章與傳回的型別

`isEmpty(<string>)`

傳回布林值。

## 範例

`isEmpty("")`

傳回true。

`isEmpty("Hello World")`

傳回false。
