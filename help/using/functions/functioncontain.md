# contain {#contain}

檢查第二個參數字串是否包含在第一個參數字串中。

## 類別

String

## 函式語法

`contain(<parameters>)`

## 參數

* 字串

## 簽名和傳回的類型

`contain(<string>,<string>)`

傳回布林值。

## 範例

`contain("rowing is great', "great")`

傳回true。
