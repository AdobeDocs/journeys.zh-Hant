# round {#round}

將最接近的整數值傳回至引數，並將四捨五入至正無窮大。

## 類別

數學

## 函式語法

`round(<parameters>)`

## 參數

* 小數點
* 整數

## 簽名和傳回類型

`round(<decimal>)`

`round(<integer>)`

傳回整數。

## 範例

`round(3.14)`

返回3。

`round(3.54)`

返回4。

`round(-3.14)`

返回-3。

`round(3)`

返回3。
