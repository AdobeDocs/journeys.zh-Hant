# 圓 {#round}

返回與參數最接近的整數值，並將捨入到正無窮大。

## 類別

數學

## 函式語法

`round(<parameters>)`

## 參數

* 小數
* 整數

## 簽名和返回的類型

`round(<decimal>)`

`round(<integer>)`

返回整數。

## 範例

`round(3.14)`

返回3。

`round(3.54)`

返回4。

`round(-3.14)`

返回–3。

`round(3)`

返回3。
