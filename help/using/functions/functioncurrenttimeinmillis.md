# currentTimeInMillis {#currentTimeInMillis}

傳回新紀元的目前時間（以毫秒為單位）。

## 類別

日期

## 函式語法

`currentTimeInMillis()`

## 參數

此函式不使用參數。

## 簽名和傳回類型

`currentTimeInMillis()`

傳回整數。

## 範例

`currentTimeInMillis()`

傳回&quot;1544712617131&quot;。
