# currentTimeInMillis {#currentTimeInMillis}

傳回目前時間（以Epoch毫秒為單位）。

## 類別

Date

## 函式語法

`currentTimeInMillis()`

## 參數

This function uses no parameters.

## 簽名和返回類型

`currentTimeInMillis()`

Returns an integer.

## 範例

`currentTimeInMillis()`

傳回&quot;1544712617131&quot;。
