# isNotEmpty {#isNotEmpty}

如果引數中的字串不是空的，則傳回true。

## 類別

字串

## 函式語法

`isNotEmpty(<parameters>)`

## 參數

* 字串

## 簽章與傳回的型別

`isNotEmpty(<string>)`

傳回布林值。

## 範例

`isNotEmpty("")`

傳回false。

`isNotEmpty("hello")`

傳回true。
