# 不為空 {#isNotEmpty}

如果參數中的字串不為空，則返回true。

## 類別

字串

## 函式語法

`isNotEmpty(<parameters>)`

## 參數

* 字串

## 簽名和返回的類型

`isNotEmpty(<string>)`

返回布爾值。

## 範例

`isNotEmpty("")`

返回false。

`isNotEmpty("hello")`

返回true。
