# toTimeZone {#toTimeZone}

將字串值轉換為時區。

## 類別

轉換

## 函式語法

`toTimeZone(<parameter>)`

## 參數

| 參數 | 說明 |
|--- |--- |
| 字串 | 字串值必須包含區域ID。 可以是欄位參考或運算式 |

## 簽名和返回的類型

`toTimeZone(<string>)`

傳回時區。

## 範例

`toTimeZone("UTC")`

傳回UTC。
