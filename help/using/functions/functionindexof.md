# indexOf {#indexOf}

檢查第一個引數字串是否以特定字串結尾（第二個引數字串）。 如果找不到字串，則傳回–1。

## 類別

字串

## 函式語法

`indexOf(<parameters>)`

## 參數

| 參數 | 類型 |
|-----------|------------------|
| 字串 | 字串 |
| 指定值 | 字串 |

## 簽章與傳回的型別

`indexOf(<string>,<string>)`

傳回整數。

## 範例

`indexOf("Hello", "l")`

傳回2。

說明：

在「Hello」中，「l」的第一次出現位置為位置2。
