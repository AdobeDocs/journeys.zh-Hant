# indexOf {#indexOf}

檢查第一個引數字串是否以特定字串（第二個引數字串）結尾。 如果找不到字串，則返回-1。

## 類別

String

## 函式語法

`indexOf(<parameters>)`

## 參數

| 參數 | 類型 |
|-----------|------------------|
| 字串 | String |
| 指定值 | String |

## 簽名和傳回的類型

`indexOf(<string>,<string>)`

傳回整數。

## 範例

`indexOf("Hello", "l")`

返回2。

說明：

在&quot;Hello&quot;中，第一次出現&quot;l&quot;是位置2。
