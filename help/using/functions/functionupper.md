# 上 {#upper}

傳回參數的大寫版本。

## 類別

字串

## 函式語法

`upper(<parameters>)`

## 簽名和傳回的類型

`upper(<string>)`

傳回字串。

## 範例

`upper("b")`

傳回&quot;B&quot;。
