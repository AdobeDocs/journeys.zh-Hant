# upper {#upper}

傳回引數的大寫版本。

## 類別

字串

## 函式語法

`upper(<parameters>)`

## 簽章與傳回的型別

`upper(<string>)`

傳回字串。

## 範例

`upper("b")`

傳回「B」。
