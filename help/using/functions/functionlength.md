# 長度 {#length}

返回參數中字串表達式的字元數。

## 類別

字串

## 函式語法

`length(<parameters>)`

## 參數

* 字串

## 簽名和返回的類型

`length(<string>)`

返回整數。

## 範例

`length("Hello World")`

返回11。
