# length {#length}

傳回參數中字串運算式的字元數。

## 類別

String

## 函式語法

`length(<parameters>)`

## 參數

* 字串

## 簽名和傳回的類型

`length(<string>)`

傳回整數。

## 範例

`length("Hello World")`

返回11。
