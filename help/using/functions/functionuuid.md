# UUID {#uuid}

產生隨機UUID （通用唯一識別碼）。

## 類別

字串

## 函式語法

`uuid()`

## 參數

此函式不需要引數。

## 簽章與傳回的型別

`uuid()`

傳回字串。

## 範例

`uuid()`

傳回「79e70b7f-8a85-400b-97a1-9f9826121553」。s
