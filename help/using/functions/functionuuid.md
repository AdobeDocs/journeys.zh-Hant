# uid {#uuid}

產生隨機UUID（通用唯一ID識別碼）。

## 類別

String

## 函式語法

`uuid()`

## 參數

此函式不需要參數。

## 簽名和傳回的類型

`uuid()`

傳回字串。

## 範例

`uuid()`

傳回&quot;79e70b7f-8a85-400b-97a1-9f9826121553&quot;。s
