# lastIndexOf {#lastIndexOf}

傳回第二個參數最後一次出現的位置（在第一個參數中）。 如果沒有匹配，則返回-1。

## 類別

字串

## 函式語法

`lastIndexOf(<parameters>)`

## 參數

| 參數 | 類型 |
|-----------|------------------|
| 字串 | 字串 |
| 指定值 | 字串 |

## 簽名和傳回的類型

`lastIndexOf(<string>,<string>)`

傳回整數。

## 範例

`lastIndexOf("Hello, "l"))`

返回3。

說明：

在&quot;Hello&quot;中，最後一次出現&quot;l&quot;是位置3。
