# 預覽裝載 {#concept_jgf_4yk_4fb}

有效負載預覽可讓您驗證有效負載定義。

>[!NOTE]
>
>對於系統產生的事件，當您建立事件時，在檢視裝載預覽之前，請先儲存事件並重新開啟它。 需要執行此步驟，才能在裝載中產生事件ID。

1. 按一下 **[!UICONTROL View Payload]** 圖示來預覽系統預期的裝載。

   ![](../assets/journey13.png)

   您可以注意到已顯示選取的欄位。

   ![](../assets/journey14.png)

1. 檢查預覽以驗證有效負載定義。

1. 接著，您可以將裝載預覽與事件傳送的負責人共用。 此裝載可協助他們設計推送至的事件設定 [!DNL Journey Orchestration]. 請參閱[此頁面](../event/additional-steps-to-send-events-to-journey-orchestration.md)。
